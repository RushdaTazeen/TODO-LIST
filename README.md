Project Description

The To-Do List application is a desktop-based task management system developed using Python and Tkinter. It helps users organize their daily tasks efficiently by allowing them to add, delete, and mark tasks as completed. Tasks are stored in a JSON file to ensure data persistence even after closing the application.

Features

- Add new tasks
- Delete selected tasks
- Mark tasks as completed
- Clear all tasks
- Automatic task saving using JSON
- Scrollable task list
- Simple and user-friendly graphical interface

Technologies Used

- Python
- Tkinter
- JSON

How to Run

1. Install Python on your system.
2. Download or clone the project.
3. Open the project folder in VS Code.
4. Open the terminal.
5. Run the following command:

python todo.py

Output

The application opens a graphical window where users can:

- Enter tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Save tasks automatically
